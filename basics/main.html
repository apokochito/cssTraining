CSS is not a programming language, it's a style sheet language

1. Anatomy of a CSS ruleset

Example

This is a pharagraph

                -- HTML --
                <p> This is a pharagraph </p>

                -- CSS --
                p {
                    color: red;
                  }

                p → Selector → HTML element
                color → Property
                red → Property value
                color: red; → Declaration → Rule

Description

The whole structure is called a ruleset (The term ruleset is often referred to as just rule).
Each ruleset must be wrapped in curly braces. ({})

2. Selecting multiple elements

Example

  • First element of the list
  • Second element of the list
  1. First element
  2. Second element
                -- HTML --
                <ul>
                    <li>First <span>element</span> of the list</li>
                    <li>Second <span>element</span> of the list</li>
                <ul>

                <ol>
                    <li>First <span>element</span> of the list</li>
                    <li>Second <span>element</span> of the list</li>
                <ol>

                -- CSS --
                div ul li span {
                    color: blue;
                }

                div ol li, span {
                    color: purple;
                }

Description

You can select multiple elements and apply a single ruleset to all of them. (ordered list)
You can navigate some elements to select only the one you need to modify. (unordered list)

3. Different types of selectors

Example

This is my p element

Description

                -- HTML --
                <p id="this-is-my-id"> This is my p element </p>

                -- CSS --
                #this-is-my-id {
                    color:green;
                }

                Element selector → <p>
                ID selector → #my-id
                Class selector → .my-class
                Attribute selector → img[src]
                Pseudo-class selector → a:hover

4. Fonts and text

Example

                -- HTML --
                We add this line <link href="https://fonts.googleapis.com/css?family=Open+Sans" rel="stylesheet"> into head tag

                -- CSS --
                html {
                    font-size: 10px; /* px means "pixels": the base font size is now 10 pixels high  */
                    font-family: "Open Sans", sans-serif; /* this should be the rest of the output you got from Google fonts */
                }

Description

Link tag links your page to a style sheet that loads the Open Sans font family with your webpage.
Anything in CSS between /* and */ is a CSS comment.
