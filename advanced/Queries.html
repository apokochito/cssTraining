CSS is not a programming language, it's a style sheet language

Media Queries: Browser-size specific CSS

Example

                -- HTML --
                <div id="media"></div>

                -- CSS --
                #media {
                    background-color: aquamarine;
                    height: 30px;
                    width: 1000px;
                }
                
                @media screen and (max-width: 2000px) {
                    #media { width: 10% }
                }

Description

They're used to target styles at specific media, such as screen or print, have already been covered.
Screen-based media of a certain size.
