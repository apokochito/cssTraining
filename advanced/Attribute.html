CSS is not a programming language, it's a style sheet language

Attribute Selectors: With the attribute

Example

I do HTML

                -- HTML --
                <p>I do <abbr title="Hypertext Markup Language">HTML</abbr></p>

                -- CSS --
                abbr[title] {
                  border-bottom: 1px dotted #ccc
                }

Description

Attribute selectors allow you to style an element’s box based on the presence of an HTML attribute or of an attribute’s value.
This basically says “shove a dotted line underneath all abbreviations with a title attribute”.

Attribute Selectors: With the attribute and its value

Example

                -- HTML --
                <input type="text" id="fid" name="fname">

                -- CSS --
                input[type=text] {
                  width: 200px;
                }

Description

You can further specify that you want to style something with a specific attribute value.
This example will apply a width of 200 pixels only to input elements that are specified as being text boxes.
