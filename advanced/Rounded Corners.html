CSS is not a programming language, it's a style sheet language

Rounded Corners: Border radius?

Example

                -- HTML --
                <div id="marilyn"></div>

                -- CSS --
                #marilyn {
                  background: #233;
                  width: 100px;
                  height: 100px;
                  border-radius: 20px;
                }

Description

The border-radius property can be used to add a corner to each corner of a box.

Rounded Corners: Multiple values

Example

                -- HTML --
                <div id="monroe"></div>

                -- CSS --
                #monroe {
                  background: #233;
                  width: 100px;
                  height: 100px;
                  border-radius: 6px 12px 18px 24px;
                }

Description

Radius can also be used to target specific corners.

Rounded Corners: Ellipses

Example

                -- HTML --
                <div id="nanoo"></div>

                -- CSS --
                #nanoo {
                  background: #233;
                  width: 100px;
                  height: 150px;
                  border-radius: 50px/100px;
                  border-bottom-left-radius: 50px;
                  border-bottom-right-radius: 50px;
                }

Description

You can specify different horizontal and vertical radiiii by splitting values with a “/”.
