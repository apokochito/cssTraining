CSS Transitions

Example

This is my link 
                -- HTML --
                <a id="first">This is my link</a>

                -- CSS --
                a:link {
                    transition: all .5s linear 0;
                    color: hsl(36,50%,50%);
                }
                a:hover {
                    color: hsl(36,100%,50%);
                }

Description

Transitions allow you to easily animate parts of your design without the need for the likes of JavaScript.
The transition property here is saying it wants all properties transitioned over half a second in a linear fashion with no delay.

CSS Transitions: Targeting specific properties

Example

This is my link 
                -- HTML --
                <a id="second">This is my link</a>

                -- CSS --
                div div #second {
                  color: rgba(235,10,160,.5);
                  transition: .5s;
                  transition-property: color;
                }
              
                div div #second:hover {
                  color: rgba(255,0,0,1);
                }

Description

While “all” can be used in transition-property (or transition), you can tell a browser only to transition certain properties, by simply plonking in the property name you want to change.
If you want to target more than one property (without using “all”), you can offer up a comma-separated list with transition-property.
