CSS is not a programming language, it's a style sheet language

CSS Transitions

Example

This is my link 
                -- HTML --
                <a>This is my link</a>

                -- CSS --
                a:link {
                    color: hsl(36,50%,50%);
                }
                a:hover {
                    color: hsl(36,100%,50%);
                }

Description

Transitions allow you to easily animate parts of your design without the need for the likes of JavaScript.
