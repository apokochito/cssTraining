Selectors: Adjacent selectors

Example

Hey

You

Man

        -- HTML --
        <h5>Hey</h5>
        <p>You</p>
        <p>Man</p>
        
        -- CSS --
        h5 + p { font-weight: bold }

Description

A plus sign (“+”) is used to target an adjacent sibling of an element, essentially, something immediately following something.
