CSS is not a programming language, it's a style sheet language

Transformations: Rotating

Example

                -- HTML --
                <div id="note"></div>

                -- CSS --
                #note {
                    width: 300px;
                    height: 300px;
                    background: hsl(36,100%,50%);
                    transform: rotate(-10deg);
                }

Description

Manipulating a box over two dimensions, transform can be used to rotate, skew, scale and translate a box and its contents.

Transformations: Skewing

Example

                -- HTML --
                <div id="note2"></div>

                -- CSS --
                #note2 {
                    width: 300px;
                    height: 300px;
                    background: hsl(36,100%,50%);
                    transform: skew(20deg,10deg);
                }

Description

Skewing allows you to tip the horizontal and vertical.

Transformations: Scaling

Example

                -- HTML --
                <div id="note3"></div>

                -- CSS --
                #note3 {
                    width: 300px;
                    height: 300px;
                    background: hsl(36,100%,50%);
                    transform: scale(2);
                }

Description

This will multiply the size by two. You can use any positive number, including a value less than “1”, such as “0.5”, if you want to use a shrink ray.
You can also scale the horizontal and vertical dimensions separately.

Transformations: Translating

Example

                -- HTML --
                <div id="note4"></div>

                -- CSS --
                #note4 {
                    width: 300px;
                    height: 300px;
                    background: hsl(36,100%,50%);
                    transform: translate(100px,200px);
                }

Description

You can shift a box horizontally and vertically using transform: translate.
Similar to position: relative; left: 100px; top: 200px;, this will move a box 100 pixels to the right and 200 pixels down.

Yout can combine all above also, for ex. transform: rotate(-10deg) scale(2); / transform: matrix(2,-0.35,0.35,2,0,0);

Transformations: Origin

Example

                -- HTML --
                <div id="note5"></div>

                -- CSS --
                #note5 {
                    width: 300px;
                    height: 300px;
                    background: hsl(36,100%,50%);
                    transform-origin: 0 0;
                }

Description

This is the origin of the transformation.
The object will move from the origin reference.
