CSS is not a programming language, it's a style sheet language

CSS Backgrounds: Multiples

Example

                -- HTML --
                <div id="div-background-tag"></div>

                -- CSS --
                #div-background-tag {
                    height: 240px;
                    width: auto;
                    background-image: url("https://encrypted-tbn0.gstatic.com/images?q=tbn:ANd9GcSeEZNxsTzbHuDVkRbJPwt-voHJ2sJ8St3-JQ&usqp=CAU"),
                    url("https://encrypted-tbn0.gstatic.com/images?q=tbn:ANd9GcQVnu4gGPDX8d8lnFbrgsJ-NPfUHkRotW3oHQ&usqp=CAU"),
                    url("https://encrypted-tbn0.gstatic.com/images?q=tbn:ANd9GcTle1cN5zC4Ljy_A0jFO61eFSy8LjpRMXfMuA&usqp=CAU");
                    background-position: left, center, right;
                    background-repeat: no-repeat, no-repeat, no-repeat;
                }

Description

CSS3 allows you to apply multiple background images to a single box by simply putting image locations in a comma-separated list.

CSS Backgrounds: Size

Example

                -- HTML --
                <div id="div-background"></div>

                -- CSS --
                #div-background {
                  height: 240%;
                  width: auto;
                  background-image: url("https://encrypted-tbn0.gstatic.com/images?q=tbn:ANd9GcSyBv42ua_scB09Ule9eHkJVHZo7FQa9zp1Eg&usqp=CAU");
                  background-size: contain;
                  display: list-item;
                  /*background-size: cover;*/
               }

Description

The background-size property allows you to stretch or compress a background image.

CSS Backgrounds: Origin

Example

                -- HTML --
                <div id="div-back"></div>

                -- CSS --
                #div-back {
                  height: 200px;
                  width: 200px;
                  border: 10px solid rgba(0,255,0,.5);
                  padding: 20px;
                  background: url("https://en.owl.institute/system/course_management/file/images/files/000/000/288/240-240/radial_gradient_diagonal_blau_rot.png?1505827421") 0 0 no-repeat;
                  background-origin: padding-box;
                  /*background-origin: border-box; */
                }

Description

It will cause the origin of the background image to be tucked up inside the border.
