CSS is not a programming language, it's a style sheet language

Gradients: Linear

Example

                -- HTML --
                <div id="linear"></div><div id="linear2"></div>

                -- CSS --
                #linear {
                  width: 200px;
                  height: 200px;
                  background: linear-gradient(yellow, red);
                }

                #linear2 {
                  width: 200px;
                  height: 200px;
                  margin: 2px;
                  padding: 0;
                  background: linear-gradient(to bottom right, orange, red);
                }

Description

For an even spread of two colors, fading from one at the top to another at the bottom.
Or one corner, Or any angle that tickles your fancy.

Gradients: Radial

Example

                -- HTML --
                <div id="radial"></div><div id="radial2"></div>

                -- CSS --
                #radial {
                  width: 200px;
                  height: 200px;
                  margin: 2px;
                  padding: 0;
                  background: radial-gradient(at top left, yellow, green);
                }
              
                #radial2 {
                  width: 200px;
                  height: 200px;
                  margin: 2px;
                  padding: 0;
                  background: radial-gradient(circle closest-side, yellow, green);
                }

Description

Radial gradients, with one color starting from a central point and fading to another color.

Gradients: Color Stops

Example

                -- HTML --
                <div id="colors"></div><div id="colors2"></div>

                -- CSS --
                #colors {
                  width: 200px;
                  height: 200px;
                  margin: 2px;
                  padding: 0;
                  background: linear-gradient(red 0%, green 33.3%, blue 66.7%, black 100%)
                }
              
                #colors2 {
                  width: 200px;
                  height: 200px;
                  margin: 2px;
                  padding: 0;
                  background: radial-gradient(#06c 0, #fc0 50%, #039 100%)
                }

Description

You can specify exactly where in the gradient each color kicks in, straight after each color, starting at “0” and ending at “100%” (although lengths can also be used).

Gradients: Repeating

Example

                -- HTML --
                <div id="repeat"></div><div id="repeat2"></div>

                -- CSS --
                #repeat {
                  width: 200px;
                  height: 200px;
                  margin: 2px;
                  padding: 0;
                  background: repeating-linear-gradient(white, black 15px, white 30px);
                }
              
                #repeat2 {
                  width: 200px;
                  height: 200px;
                  margin: 2px;
                  padding: 0;
                  background: repeating-radial-gradient(black, black 15px, white 15px, white 30px);
                }

Description

A single gradient will fill a box with the previous methods but you can use “repeating-linear-gradient” and “repeating-linear-gradient” to build on the color stops and, well, repeat the gradient.
