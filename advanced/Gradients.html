Gradients: Radial

Example

                -- HTML --
                <div id="linear"></div><div id="linear2"></div>

                -- CSS --
                #linear {
                  width: 200px;
                  height: 200px;
                  background: linear-gradient(yellow, red);
                }

                #linear2 {
                  width: 200px;
                  height: 200px;
                  margin: 2px;
                  padding: 0;
                  background: linear-gradient(to bottom right, orange, red);
                }

Description

For an even spread of two colors, fading from one at the top to another at the bottom.
Or one corner, Or any angle that tickles your fancy.
